# Bank_Loan_Prediction
Project on Banking Domain
 this is my first machine learning project 

Production link - https://bank-loan-prediction-m3fc.onrender.com

 #Overview
This project predicts whether a bank loan application should be approved or rejected using a Machine Learning model.
The model has been deployed using Flask with an interactive web interface built using HTML, CSS, and Bootstrap.

It helps financial institutions analyze customer details such as age, job, education, loan history, and other attributes to make data-driven lending decisions.
